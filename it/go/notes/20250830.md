## Install dependencies

### Enable live reload

- [Air](https://github.com/air-verse/air)

### Golang migrate

- [Install Golang migrate](https://github.com/golang-migrate/migrate/blob/master/cmd/migrate/README.md)

Document:

- [golang 数据库迁移工具 golang-migrate 使用](https://blog.csdn.net/doyzfly/article/details/121096806)
- [数据库版本管理利器：Golang Migrate 入门到实践](https://blog.csdn.net/m0_57836225/article/details/149550484)

Download and Install:

[Download latest version](https://github.com/golang-migrate/migrate/releases)

```shell
sudo dpkg -i migrate.linux-amd64.deb
migrate -version # 4.18.3
```

## Setting up the project

### Initialize a new Go module

```sh
mkdir rest-api-go-gin
cd rest-api-go-gin
go mod init rest-api-go-gin
air init
```

Project structure should look like this:

```shell
rest-api-go-gin
├── cmd
│   ├── api
│   │   ├── main.go
│   ├── migrate
│   │   ├── main.go
│   │   └── migrations
├── internal
│   ├── database
│   ├── env
```
