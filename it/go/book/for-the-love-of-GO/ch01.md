# Testing times
## Creating a new project
```shell
mkdir -p workspace/go/loveOfGo/ch01/calculator
cd workspace/go/loveOfGo/ch01/calculator
go mod init calculator

touch calculator.go calculator_test.go
```
*calculator.go*
```go
package calculator

func Add(a, b float64) float64 {
	return a + b
}
```

*calculator_test.go*
```go
package calculator_test

import (
	"calculator"
	"testing"
)

func TestAdd(t *testing.T) {
	t.Parallel()
	var want float64 = 4
	got := calculator.Add(1, 3)
	if want != got {
		t.Errorf("want %f, got %f", want, got)
	}
}
```
Run:
```shell
go test
```

## Formatting code with `gofmt`
```sh
gofmt -d calculator.go
```
