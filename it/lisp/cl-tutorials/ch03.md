# let, let*, setq, defun, phew!
## let
```lisp
(let (var)
  (princ var))

;;
; nil


(let ((x 5) (y 3))
  (princ x)
  (terpri)
  (princ y)
  (terpri)
  (princ (+ x y)))

;;
; 5
; 3
; 8
```

## let*
```lisp
(let* ((x 5) (y x))
  (+ x y))
```

## setq
```lisp
(let (x)
  (princ x) ; nil
  (terpri)
  (setq x 5)
  (princ x) ; 5
  (+ 13 x)) ; 8
```
